Differentiation

Differentiation is used to caculate rate of change on a gradient of tangent to a curve with the function of change in x is a change in y that can be seen as (dy/dx). For example, the change of velocity in acceleration

(Sacchetti, Monagan, 2023)

Rules Formula Explain Example 1 Example 2
Constant Rule d/dx *c = 0 Any number(indicated with "c") without a x axis is equal to zero 10 ==> 0 69 ==> 0
Power Rule d/dx * xⁿ = nxⁿ⁻¹ When x axis has power, time number of power to x and remove a power from x x² ==> 2x 2x⁴ ==> 8x³
Trigonometric Rules d/dx sin(x) = cos(x) Sin turns into cos sin(5) ==> cos(5) sin(45) ==> cos(45)
Trigonometric Rules d/dx cos(x) = -sin(x) Cos turns into -sin cos(5) ==> -sin(5) cos(45) ==> -sin(45)
Exponential Rules d/dx b^x = b^x In(b) Exponential turns into log d/dx b^5 = b^5In(b) d/dx b^45 = b^45In(b)
Logarithmic Rules d/dx ln(x) = 1/x Log turns into fraction x of 1 d/dx ln(5) = 1/5 d/dx ln(45) = 1/45
